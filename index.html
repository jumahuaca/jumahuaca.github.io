Jumahuaca — El proyecto

Jumahuaca es una idea ambiciosa que intenta construir una comunidad preocupada por las arquitecturas testeables. El proyecto se puede resumir en los siguientes componentes ...

El Framework

Un framework en Java con diferentes utilidades para probar distintos tipos de aplicaciones Java Enterprise. Tests genéricos, matchers, clases utilitarias, mocks configurados. Enterate de más entrando acá.

Los ejemplos

Ejemplos de uso del framework en casos de uso típicos de aplicaciones Java Enterprise como imágenes Docker. Los ejemplos van desde acceso a datos con jdbc hasta serialización en json con distintos frameworks. Revisá la lista completa acá.

El blog

Un blog donde compartimos nuestras opiniones sobre el proyecto y el testing en general. También revisaremos libros, sitios y papers que traten sobre la temática. Dejanos saber tu opinión acá.

La documentación

Toda la documentación del proyecto. Cómo empezar, cómo descargar el framework como dependencia, qué tener en cuenta para usar los ejemplos, los principios de testing utilizados y mucho más. Empezá a leer desde acá.

El nombre Jumahuaca nació en un viaje al Norte Argentino. La historia completa acá.
